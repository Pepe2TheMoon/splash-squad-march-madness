# Splash Squad March Madness Warroom v5

Self-debating AI agent system for March Madness predictions — Monte Carlo sims, Bayesian inference, sentiment analysis, continuous learning, and live web intelligence.

## Features
- War Room: 4 AI agents debate matchups with live web search
- Live Intel Feed: Continuous news, scores, rankings stream
- Monte Carlo Sims: Up to 50K runs with 18 tunable variables
- Continuous Training: Auto-trains on 55+ historical games
- Theoretical Lab: 10 archetypes, agent debates sims
- Knowledge Brain: Lessons feed into all agents
- Bracket Day: Tournament countdown and bracket generation

## Quick Start
```
git clone https://github.com/Pepe2TheMoon/splash-squad-march-madness.git
cd splash-squad-march-madness
npm install
npm run dev
```
